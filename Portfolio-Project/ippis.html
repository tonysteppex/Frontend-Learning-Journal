IPPIS Website Redesign

IPPIS Website Redesign is a project that involved a total overhaul of the Integrated Personnel and Payroll Information System (IPPIS) website. The goal was to improve the user experience, enhance the visual design, and ensure that the website is accessible and user-friendly for all users.

Role and Tasks

  • Design Pages: Designed the website pages from the home page to the contact pages and all the pages in between.
  • Responsiveness: Ensured responsiveness across desktop and mobile screens to ensure a great user experience across every breakpoint.
  • Prototyping: Developed high-fidelity interactive prototypes in Figma for stakeholder testing.

Key Accomplishments

  • Successfully designed a modern, user-friendly interface that enhances the overall user experience.
  • Implemented a responsive design that provides an optimal viewing experience across a wide range of devices.
  • Created interactive prototypes that facilitated stakeholder feedback and iterative improvements.
  • Delivered the complete design system handover for the development team.
