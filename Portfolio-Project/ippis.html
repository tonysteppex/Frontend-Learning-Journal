IPPIS Website Redesign

Project Design Engineer

2025

The system handles payroll for over 1.1 million federal employees, but the primary website was outdated, difficult to navigate, and lacked mobile support. I designed a modern, accessible interface focusing on user self-service. This included building clear workflows for new staff enrollment, an automated complaints ticketing system, and live chat support.

Key Accomplishments

  • Modernized outdated, legacy interfaces into clean, responsive layouts accessible on mobile devices.
  • Streamlined navigation for complex administrative workflows, significantly reducing friction for end users.
